# Changelog

All notable changes to this project will be documented in this file.

## [0.31.0] - 2026-02-16

### Added
- **Model Context Protocol (MCP)**: Connect ZakatFlow to ChatGPT/Claude for context-aware answers about strict Zakat rulings and portfolio analysis.
- **Open Source**: Repository is now public under AGPL-3.0 license.
- **Deep Link Reports**: Generate base64-encoded report links via MCP tools to quickly share or save calculation states.

### Changed
- **Waitlist Removed**: Application is now fully open for public use.

## [0.30.0] - 2026-02-16

### Added
- **Real-Time Methodology Engine**: Runs actual Zakat logic in the browser for immediate feedback.
- **Interactive Comparison Tool**: Compare "Theory" vs "Practice" (e.g., Ahmed Case Study).
- **Case Studies**: Standardized profiles to demonstrate rule differences across Madhabs.

### Changed
- **Methodology Explorer**: Enhanced UI with dynamic calculation visualization.

## [0.29.0] - 2026-02-15

### Added
- **Financial Statement Hero**: Realistic report card design for landing page.
- **Phosphor Icons**: Complete migration for unified visual language.
- **Scroll UX**: Consistent navigation cues.

### Changed
- **Footer**: Simplified scholarly citations.
- **Privacy Badges**: Clearer session encryption status.

## [0.28.0] - 2026-02-15

### Added
- **AGPL-3.0 License**: Codebase is fully open source.
- **Two-Tier Encryption**: Managed vs Sovereign key management options.
- **Plaid Encryption**: Bank tokens encrypted with user-derived keys.

### Fixed
- **Upload Classification**: Improved AI mapping for statements.
- **Bank Sync**: Better category detection for Plaid accounts.
